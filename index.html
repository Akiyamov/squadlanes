Capture points and RAAS lanes are automatically extracted from Squad maps. Unfortunately, naming inconsistencies in the map elements and a lack of information on Unreal Engine 4's serialized object format require some manual fixes on the automatically extracted data. Some map layers displayed here might be incorrect.

If you want to explore or contribute to this project, feel free to check out our repository on GitHub: w4rum/squadlanes

Known Issues:
  • Some capture point names don't exactly match their in-game equivalent, e.g., "Mine Entrance 8" vs. "Mine Entrance".
  • Some capture points belong to too many / too few / the wrong lanes.
  • Some capture points are placed at an incorrect depth.

If you spot any specific errors, please head over to our GitHub repository and open an issue! That helps us get an overview.
